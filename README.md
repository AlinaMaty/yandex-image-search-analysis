# 📊 Анализ поисковых запросов в Яндекс Картинках

## 📌 О проекте

Данный проект выполнен в рамках тестового задания на позицию **Продуктового / BI-аналитика**.

Цель анализа — исследовать поведение пользователей сервиса поиска изображений и проверить гипотезу о различиях в интересах пользователей в зависимости от платформы:

* 🖥 Desktop
* 📱 Mobile (touch)

---

## 🎯 Цель исследования

Проверить гипотезу:

> Пользовательские интересы и поведение в поиске изображений отличаются между мобильными устройствами и десктопом.

---

## 🗂 Данные

Использован датасет с поисковыми запросами пользователей Яндекс Картинок за несколько недель.

**Содержит:**

* текст запроса
* тип платформы (desktop / touch)
* временную метку

---

## 🧩 Задачи анализа

В рамках проекта были решены следующие задачи:

1. Определён временной диапазон предоставленных данных
2. Рассчитано количество запросов со словом **"ютуб"** по платформам
3. Выведены **топ-10 самых популярных запросов** для каждой платформы
4. Проанализирована **суточная динамика пользовательской активности**
5. Выделены **тематики запросов**, отличающиеся между платформами

## 🔍 Основные результаты и инсайты

### 1. Различие пользовательского поведения

* Пользователи **mobile** и **desktop** демонстрируют различающиеся паттерны поиска
* Это подтверждает исходную гипотезу

---

### 2. Популярность запроса "ютуб"

* Запросы со словом *"ютуб"* чаще встречаются на **mobile**
* Возможное объяснение: мобильные устройства чаще используются для потребления видеоконтента

---

### 3. Различия в популярных запросах

**Desktop:**

* более целевые и "практичные" запросы
* выше доля информационного поиска

**Mobile:**

* больше развлекательных и визуальных запросов
* выше доля коротких и быстрых поисков

---

### 4. Суточная активность

* **Desktop**:

  * пик в рабочее время
* **Mobile**:

  * более равномерное распределение
  * повышенная активность вечером

📌 Возможное объяснение:

* Desktop используется в рабочем контексте
* Mobile — в течение всего дня, включая досуг

---

### 5. Тематические различия

Выделены категории запросов, отличающиеся по платформам:

* 📱 Mobile:

  * развлечения
  * мемы
  * визуальный контент

* 🖥 Desktop:

  * работа / обучение
  * информационные запросы
